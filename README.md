# Abhishek-0005 Monorepo

This repository now includes a full-stack Expense Tracker under `expense-tracker/`.

- See `expense-tracker/README.md` for complete setup instructions.
- Technologies: React + Vite (frontend), NestJS + Prisma + PostgreSQL (backend).
- Docker Compose is provided to run Postgres and backend.
