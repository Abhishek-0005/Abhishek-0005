## 🎮 Snake Game (Vite + React)
A classic Snake game is included in this repo inside the `snake-game/` subfolder.

Run locally:

```
cd snake-game
npm install
npm run dev
```

Build:
```
npm run build
```

The game includes keyboard and mobile controls, score + high score, pause/resume, restart, and responsive UI with light/dark theme.
