# Calculator

Simple, responsive, and accessible calculator web app.

Features
- AC (clear), delete (⌫), digits 0–9, decimal, +, −, ×, ÷, %, negate (+/−), equals
- Keyboard support: 0–9, ., +, -, *, /, %, Enter/Return (=), Backspace (delete), Escape (AC)
- Prevents invalid inputs (e.g., multiple decimals), handles division-by-zero with error state cleared by AC
- Proper number formatting; preserves trailing decimal while typing
- Responsive layout with focus outlines and ARIA labels

Getting started
- Open calculator/index.html in a browser. No build step required.

Files
- index.html – markup and accessible structure
- styles.css – responsive styling
- script.js – logic and keyboard handling
- tests/test.html – in-browser smoke tests
